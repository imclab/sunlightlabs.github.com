Bills in Congress

Government Printing Office

The Government Printing Office (GPO) publishes a collection of data pertaining to bills passing through Congress. For each bill, GPO publishes several data files containing the full text of legislation in plaintext, XML, and PDF format. GPO also publishes "MODS" metadata accompanying each bill, and crawl-able sitemaps that can be used to find each piece of content in the order it has been changed.

These files contain some light metadata related to versioning of bills, but lack substantial information about a bill's history, description, or status -- this is what the Library of Congress' THOMAS system specializes in. The best, though unofficial, source for obtaining that information in bulk is GovTrack.us.

Here, you can download zip files per Congress for the plaintext and XML versions of each bill, along with its MODS metadata, going back to the 103rd Congress (1993-1994).

Filename Pattern

Bill (plaintext)
[bill_version_id].htm
Bill (XML)
[bill_version_id].xml
MODS metadata
[bill_version_id].mods.xml
bill_version_id
[type][number]-[congress]-[version_code]
type
One of: [hr, hres, hjres, hcres, s, sres, sjres, scres]
number
Bill number.
congress
Number of Congress. (e.g. 111, 112)
version_code
GPO-assigned abbreviation for version type.

Votes in Congress

House of Representatives

The Clerk of the House of Representatives publishes roll call votes in XML, within minutes of the vote being taken. The latest votes are listed by year on HTML pages, which link to XML files for individual votes.

Here, you can download zip files containing all House roll call votes for a given year, back to 1990.

Senate

The Senate publishes roll call votes in XML, within an hour of the vote being taken. The latest votes are organized by Congress and session in HTML and XML. These lists link to individual votes, published in HTML and XML.

Here, you can download zip files containing all Senate roll call votes for a given year, back to 1989.
